# Training models — Estudos de Treinamento no SageMaker Studio (JupyterLab)

Esta pasta reúne os materiais de estudo e prática para o fluxo de **treinamento de modelos no Amazon SageMaker Studio (JupyterLab)**, utilizando o dataset **Employee.csv** e o algoritmo **XGBoost**.

O foco desta etapa é **entender o pipeline de Machine Learning na prática**, incluindo:

- preparação dos dados
- treinamento do modelo
- revisão de métricas e artefatos
- hyperparameter tuning (ajuste de hiperparâmetros)

> Observação: o objetivo principal é compreender o processo e o fluxo no SageMaker, não apenas executar células do notebook.

# Training models — Ordem de estudo sugerida

Esta pasta foi organizada para seguir uma sequência de aprendizado sobre treinamento de modelos na AWS (SageMaker), começando pelos **casos de uso** e avançando até o **ajuste de hiperparâmetros**.

## Ordem recomendada

1. **`aws_train_use_cases/`**  
   Entender os melhores casos de uso para treinamento na AWS.

2. **`prepare_data/`**  
   Preparar os dados para o processo de treinamento.

3. **`train_model/`**  
   Executar e revisar o treinamento do modelo.

4. **`hyperparameters_tuning/`**  
   Ajustar hiperparâmetros para otimizar o desempenho do modelo.
