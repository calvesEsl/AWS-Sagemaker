# AWS SageMaker — Estudos

Este repositório é dedicado aos meus estudos do **Amazon SageMaker** e suas principais ferramentas, reunindo anotações e demonstrações práticas com foco em aprender na prática e consolidar conhecimento.

O conteúdo percorre o ciclo de Machine Learning no SageMaker, desde:
- preparação e organização do ambiente
- importação e preparação de dados
- experimentação e treinamento de modelos
- avaliação e ajustes
- deploy e uso do modelo em produção (endpoints/serving)
- monitoramento e boas práticas

A proposta é manter um material direto e reproduzível, para reforçar meus estudos e também servir como referência para outras pessoas.
